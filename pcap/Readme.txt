This folder contains PCAP files with captured traffic generated by mobile apps. The traffic also contains additional communication of OS or other apps (noise) so it cannot be used directly for creating JA3 fingerprints.

The persons who contributed to generation of these files are the following:
- Petr Matousek
- Malombe Victor
- Matej Melus
